• Enter Registration Key - will open the YLoader "Enter Key" dialog
  • Hardware Change Log - will open the "Save hardware change history" dialog. This file is sometimes useful to diagnose changes in the local hardware fingerprint
  • Unregister - will unregister the current validation key from the computer. This command is useful when the user wants to move the YLoader to a new computer. After the key has been unregistered, a confirmation code is shown. Email this code to yloader.com for confirmation. Use this command carefully - once the key has been unregistered, it cannot be used again.
