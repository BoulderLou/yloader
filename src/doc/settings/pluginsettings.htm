The plug-ins settings page shows the list of available plug-ins and their properties. Currently, only datasource plug-ins are supported. The selected data source plug-in will be used when downloading the data.

This dialog also shows the various properties for the currently selected plug-in. These properties are read-only and they include the plug-in's capabilities, info about its developer (name, url, email) as well as general info (name, location etc) about the selected plug-in.
